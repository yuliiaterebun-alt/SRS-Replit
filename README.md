# SRS-Replit
Specification of software requirements for a platform for creating, testing, and deploying software. 
## Overwiew
Цей документ містить списифікацію вимог до програмного забезпечення (SRS) для сервісу, що спеціалізується на реалізації додатків. Система дозволяє працювати над проектами прямо в браузері, економлячи ресурси користувачів на наймі професійних розробників.
